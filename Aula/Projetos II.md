### Propriedades Logarítmicas
$$\log_b(a) = x \to b^x = a$$

$$\log_b(xy) = \log_b x + \log_b y$$

$$\log_b!\left(\frac{x}{y}\right) = \log_b x - \log_b y$$

$$\log_b(x^n) = n\log_b x$$

$$\log_b x = \frac{\log_k x}{\log_k b}$$

$$\log_b b = 1$$

$$\log_b 1 = 0$$

$$\log_b b^x = x$$

$$b^{\log_b x} = x$$
